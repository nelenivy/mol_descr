test8nv4 (M,MAX,n_molek,kol,koef,l_granutcua,r_granutcua,glybuna,kol_coced,Rad)-главная функция
	задача
		построение наилучшей мадели для прдсказания активности в рамках методда группового учета аргументов на основе k-nn метода с ограничением по радиусу
	параметры:
		M- 	метрика (значение параметра пока не влияет на результат)
		MAX - 	кол-во дискрипторов в выборке
		n_molek-	количество молекул в выборке
		kol-	количество дискрипторов, которое будет отобрано при каждом проходе
		koef-	параметр функции качества( кол-во правильно спрогнозированных молекул+koef*(кол-во отказов)
			если koef=2 то функция кочества считается по формуле (кол-во правильно спрогнозированных молекул)/(количество молекул в выборке-кол-во отказов)*100
		l_granutcua-ошибка в класификации считается допустимой если она больше значения l_granutcua
		r_granutcua-ошибка в класификации считается допустимой если она больше значения r_granutcua
		glybuna-	количество селекций МГУА
		kol_coced-количеств0 соседей
		Rad-	правило поиска ограничения по радиюсу (допустимые значения натуральные числа и 0)
			0-	нет ограничений по радиусу
			n-	радиус равен n-тому по длине ребру минимального покривающего дерева
	Вход
		y.txt -	фаил с активностью (состоит из 1 -1)
		result.txt-	матрица дискрипторов (по строкам дескрипторы по столбцам молекулы в начале каждой строки название дескриптора)
	Выход
		out.txt- 	параметры запуска и результат(count значениефункции качества  max кол-во правильно спрогнозированных молекул  neopr кол-во отказов    k количеств0 соседей    num1 135 ....  numn 227   verno кол-во процентов правидьно предсказанных мллекул в выборке
		result1.txt-	матрица дискрипторов без повторений
	пример запуска
		test8nv4('euk',547,56,6,0,-0.5,0.5,4,10,0)
claster(num_dick,kol_kl,n_molek)-главная функция
	задача
		разбиение молекул на кластеры методом минимального покрывающего дерева (МПД)
	парамеры
		num_dick-	допустимые значения натуральные числа и 0
			n-	номер дискриптора по которому строится МПД
			0- 	МПД строится по всем дискрипторам
		kol_kl- 	количество кластеров которое хотим получить
		n_molek-	количество молекул в выборке
	Вход
		y.txt -	фаил с активностью 
		result.txt-	матрица дискрипторов (по строкам дескрипторы по столбцам молекулы в начале кждой строки название дескриптора)
	Выход
		yn.txt-	фаил с активностью для n-того кластера
		clastern.txt-матрица дискрипторов для n-того кластера (!!!ПО СТРОКАМ МОЛЕКУЛЫ ПО СТОЛБЦАМ ДИСКРИПТОРЫ!!!)
		claster_inf.txt-распредиление молкул по кластерам и условие принодлежности к кластеру
